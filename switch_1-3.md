# 第三步：認識光纖線（Fiber Optic Cable）

## 什麼是光纖線？

當我們已經有：

- 交換機（Switch）
- 光模組（Optical Transceiver）

最後還需要一個媒介，才能真正把兩台設備連接起來。

這個媒介就是：

> **光纖線（Fiber Optic Cable）**

光纖線的功能就像一般網路線（RJ45），但它傳送的不是電，而是**光（Laser Light）**。

因此，資料在光纖中不是以電壓高低（0、1）傳輸，而是利用雷射光快速閃爍來表示資料。

例如：

```
交換機 A
     │
     │ 電訊號
     ▼
光模組
     │
     │ 光訊號
     ▼
═══════════════════════
      光纖線
═══════════════════════
     ▲
     │ 光訊號
光模組
     ▲
     │ 電訊號
交換機 B
```

因此：

> **光纖線就是負責傳送光訊號的高速傳輸媒介。**

---

# 光纖線長什麼樣子？

一條光纖線通常比一般網路線更細、更柔軟。

實際上，真正負責傳送光的部分非常細。

一條光纖可分成幾個主要結構：

```
──────────────────────────
外被 (Jacket)
──────────────────────────
緩衝層 (Buffer)
──────────────────────────
包層 (Cladding)
──────────────────────────
核心 (Core)
──────────────────────────
```

---

## 1. Core（核心）

Core 是光真正傳輸的地方。

材質通常是：

**高純度石英玻璃（Silica Glass）**

或特殊玻璃材料。

光就在這條非常細的玻璃裡傳播。

它的直徑非常小：

- 單模：約 9 μm（微米）
- 多模：約 50 μm 或 62.5 μm

作為比較：

一根頭髮約：

```
70 μm
```

也就是說：

> 光纖的核心甚至比人的頭髮還細。

---

## 2. Cladding（包層）

核心外面包著一層：

**Cladding（包層）**

它也是玻璃。

但是：

折射率（Refractive Index）比較低。

這樣光在核心內會發生：

> **全反射（Total Internal Reflection）**

因此光才不會跑出去。

可以想像成：

```
╔══════════════╗
║  ↘      ↗    ║
║    ↘  ↗      ║
║      ↘       ║
║    ↗  ↘      ║
║ ↗      ↘     ║
╚══════════════╝
```

光一直反射，

一路往前走。

---

## 3. Buffer（保護層）

包層外面還有：

塑膠保護層。

主要功能：

- 防止玻璃受損
- 增加柔軟度
- 避免折斷

---

## 4. Jacket（外被）

最外面就是大家看到的：

黃色

水藍色

橘色

等塑膠外皮。

除了保護之外，

最大的用途就是：

> **辨識光纖種類。**

---

# 光是如何在光纖裡傳輸？

很多人第一次接觸光纖都會以為：

雷射光就是一直直線射出去。

其實只有單模光纖比較接近這樣。

光在不同種類光纖中的傳播方式並不一樣。

---

# 單模光纖（Single-Mode Fiber，SMF）

最常見顏色：

**黃色（Yellow）**

```
═══════════════════════
黃色
═══════════════════════
```

---

## 為什麼叫 Single-Mode？

Single：

代表：

只有一種傳播模式（Mode）。

因為：

核心非常細：

約：

```
9 μm
```

光幾乎只能：

```
──────────────►
```

沿著中心前進。

因此：

- 光路單純
- 色散（Dispersion）小
- 訊號品質最好
- 可傳輸非常遠

---

## 傳輸距離

依照光模組不同：

大約可以：

- 2 km
- 10 km
- 20 km
- 40 km
- 80 km
- 120 km

甚至更遠。

因此：

大型企業、

電信商、

跨城市、

跨校區、

跨廠區

都使用：

**單模光纖。**

---

## 優點

- 傳輸距離最遠
- 頻寬最高
- 幾乎沒有模式色散（Modal Dispersion）
- 適合高速網路

---

## 缺點

因為核心太細：

光源要求很高。

因此通常需要：

**Laser（雷射）**

設備成本較高。

---

# 多模光纖（Multi-Mode Fiber，MMF）

常見顏色：

```
Aqua（水藍色）

Orange（橘色）

Magenta（部分 OM4/OM5）
```

其中：

目前資料中心最常見的是：

**Aqua（OM3 / OM4）**

---

## 為什麼叫 Multi-Mode？

因為：

Core 很粗。

通常：

```
50 μm

或

62.5 μm
```

因此：

光可以走很多路徑。

例如：

```
╔══════════════╗
║↘            ↗║
║  ↘        ↗  ║
║    ↘    ↗    ║
║      ↘       ║
║    ↗    ↘    ║
║ ↗        ↘   ║
╚══════════════╝
```

一直反射前進。

這就是：

Multiple Modes

（多種傳播路徑）

---

## 優點

核心較粗，

容易對準。

因此：

- 光模組比較便宜
- 安裝容易
- 適合短距離

因此：

機房、

企業、

實驗室、

資料中心機櫃

大量採用。

---

## 缺點

因為：

每束光走的距離不同。

有些：

```
近

有些：

遠
```

最後就會：

不同時間抵達。

這稱為：

**模式色散（Modal Dispersion）**

因此：

距離越遠，

訊號越容易失真。

---

## 傳輸距離

依速度不同：

大約：

| 速度 | 距離 |
|------|------|
| 10G | 約 300～400 公尺 |
| 40G | 約 100～150 公尺 |
| 100G | 約 70～150 公尺 |

因此：

多模主要適合：

**數百公尺內。**

---

# 光纖顏色代表什麼？

實驗室裡，

最容易辨認的方法就是：

看外皮顏色。

常見如下：

| 顏色 | 光纖種類 | 英文 | 常見用途 |
|------|----------|------|----------|
| 黃色 | 單模 | Single-Mode (SMF) | 電信商、跨樓層、跨校區、長距離 |
| Aqua（水藍） | 多模 OM3／OM4 | Multi-Mode (MMF) | 資料中心、伺服器、交換機 |
| 橘色 | 多模 OM1／OM2 | Multi-Mode (MMF) | 舊式設備、企業網路 |
| 洋紅色（Magenta） | 多模 OM4／OM5（部分廠商） | Multi-Mode (MMF) | 高速資料中心（依廠商設計） |

> **提醒：**
> 光纖外皮顏色是常見辨識方式，但仍可能因廠商、線材規格或客製化需求而有所不同。實際使用時，應搭配線材標示（如 OM3、OM4、OS2 等）確認規格。

# 光纖在資料中心中的應用

例如：

```
Server
    │
    │
 LC 光纖
    │
Switch
    │
 MPO 光纖
    │
Core Switch
```

在 AI 資料中心中：

一個 Rack（機櫃）可能有數十甚至上百條光纖。

它們負責：

- GPU 與 GPU 通訊
- 交換機互連
- 儲存設備互連
- Backbone 網路

形成整個高速網路。

---

# 光纖與銅線比較

| 項目 | 光纖 | 銅線（RJ45） |
|------|------|--------------|
| 傳輸介質 | 光 | 電 |
| 抗電磁干擾 | 幾乎不受影響 | 易受干擾 |
| 傳輸距離 | 幾百公尺到數十公里 | 約 100 公尺 |
| 可支援速度 | 10G～800G 以上 | 通常 1G～10G（特殊規格可更高） |
| 重量 | 較輕 | 較重 |
| 適用場景 | 資料中心、電信、長距離 | 一般辦公室、家庭網路 |

---

# 重點整理

- 光纖線（Fiber Optic Cable）是負責傳送光訊號的傳輸媒介，搭配交換機與光模組才能完成高速網路連線。
- 光纖內部主要由 **Core（核心）**、**Cladding（包層）**、**Buffer（保護層）** 與 **Jacket（外被）** 組成，其中 Core 是光真正傳輸的地方。
- 單模光纖（Single-Mode Fiber，SMF）核心極細，光幾乎直線傳播，適合數公里至數十公里的長距離高速傳輸，常見外皮為黃色。
- 多模光纖（Multi-Mode Fiber，MMF）核心較粗，光會以多條路徑反射前進，適合數百公尺內的短距離應用，常見外皮為 Aqua（水藍色）或橘色。
- 光纖外皮顏色可協助辨識類型，但仍應以線材標示（如 OS2、OM3、OM4、OM5）確認實際規格。
- 常見光纖接頭包括 **LC Connector**（適用於 SFP、QSFP）以及 **MPO/MTP Connector**（適用於 100G、400G、800G 等高速多芯傳輸）。
- 相較於銅線，光纖具有傳輸距離遠、頻寬高、抗電磁干擾能力強等優勢，因此已成為現代資料中心、雲端運算與 AI 叢集的主要連線方式。
